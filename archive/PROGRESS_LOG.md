## Sequential Advancement Rules

To maintain momentum and avoid drift:

1. **Daily Entry Required** — Every day of work gets an entry
2. **Checkpoint Before Next** — Mark current phase complete before starting next
3. **Blockers Escalate Immediately** — If blocked > 4 hours, document and escalate
4. **Deliverables Tracked** — Every file change recorded
5. **Agent Handoff** — End of session = handover to next agent
